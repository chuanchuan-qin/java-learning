# ☕ Java 变量详解

变量是 Java 程序中最基本的存储单元，可以理解为**内存中一块命名的数据存储区域**。每个变量都有特定的数据类型、名称和作用域。

---

## 1. 变量的声明与赋值
- 先声明变量，再赋值。
- 可以在声明时初始化，也可以在赋值时初始化。
```java
// 声明
int age;
// 赋值
age = 25;
// 声明并初始化
double salary = 8000.50;
String name = "张三";
```

- 可以在一行声明多个同类型变量（不推荐）：
```java
int a, b, c;
int x = 1, y = 2;
```

---

## 2. 变量的数据类型

Java 是强类型语言，变量必须先声明类型。变量类型分为两大类：

### 2.1 基本数据类型（8 种）

Java 内置的简单数据类型，直接存储数值。

| 类型    | 关键字       | 占用字节 | 取值范围                  | 默认值      | 示例                     |
|-------|-----------|------|-----------------------|----------|------------------------|
| 字节型   | `byte`    | 1    | -128 ~ 127            | 0        | `byte b = 100;`        |
| 短整型   | `short`   | 2    | -32,768 ~ 32,767      | 0        | `short s = 30000;`     |
| 整型    | `int`     | 4    | -2³¹ ~ 2³¹-1          | 0        | `int i = 10;`          |
| 长整型   | `long`    | 8    | -2⁶³ ~ 2⁶³-1          | 0L       | `long l = 100000L;`    |
| 单精度浮点 | `float`   | 4    | ±3.4E-38 ~ ±3.4E+38   | 0.0f     | `float f = 3.14f;`     |
| 双精度浮点 | `double`  | 8    | ±1.8E-308 ~ ±1.8E+308 | 0.0d     | `double d = 3.14159;`  |
| 字符型   | `char`    | 2    | 0 ~ 65,535（Unicode）   | '\u0000' | `char c = 'A';`        |
| 布尔型   | `boolean` | /    | `true` 或 `false`      | false    | `boolean flag = true;` |

- `long` 类型字面量建议加 `L`，`float` 必须加 `f` 或 `F`。
- `char` 表示单个字符，使用单引号，可以存中文。

### 2.2 引用数据类型

存储的是对象的内存地址（引用），不是值本身。包括：
- **类**：`String`、自定义类
- **接口**：`List`、`Runnable`
- **数组**：`int[]`、`String[]`
- **枚举**：`enum`

```java
String str = "Hello";   // 引用
int[] arr = {1,2,3};    // 数组引用
Object obj = new Object();
```

默认值为 `null`（即不指向任何对象）。

## 3. 变量的分类（按作用域）

### 3.1 局部变量（Local Variables）
- **定义位置**：方法、构造器或代码块内部。
- **特点**：
    - 必须**显式初始化**后才能使用，无默认值。
    - 生命周期随方法/代码块的执行结束而销毁。
    - 存储在栈内存中。

```java
public void test() {
    int a = 5;          // 局部变量
    if (a > 0) {
        int b = 10;     // 更小范围的局部变量
        System.out.println(b);
    }
    // System.out.println(b);  错误！b已失效
    int c;
    // System.out.println(c);  错误！c未初始化
}
```

### 3.2 实例变量（成员变量 / Instance Variables）
- **定义位置**：类中、方法外，没有 `static` 修饰。
- **特点**：
    - 属于**对象**，每个实例有一份副本。
    - 有默认值（整型为0，浮点型为0.0，字符型为'\u0000'，，布尔为false，引用为null）。
    - 随对象创建而分配空间，对象被回收时消失。
    - 存储在堆内存中。

```java
public class Person {
    String name;      // 实例变量，默认null
    int age;          // 默认0

    public void info() {
        System.out.println(name + ", " + age);
    }
}
```

### 3.3 静态变量（类变量 / Static Variables）
- **定义位置**：类中、方法外，使用 `static` 关键字。
- **特点**：
    - 属于**类**，被所有实例共享，只有一份拷贝。
    - 有默认值，同实例变量默认值相同。
    - 在类加载时初始化，程序结束时销毁。
    - 可通过 `类名.变量名` 直接访问。

```java
public class Counter {
    public static int count = 0;  // 类变量
    public Counter() {
        count++;
    }
}

// 使用：Counter.count
```

**三种变量对比速查**：

| 变量类型 | 修饰符      | 所属     | 默认值     | 生命周期   | 存储位置     |
|------|----------|--------|---------|--------|----------|
| 局部变量 | 无        | 方法/代码块 | 无，必须初始化 | 方法执行期间 | 栈        |
| 实例变量 | 无 static | 对象     | 有       | 对象存活期间 | 堆        |
| 静态变量 | static   | 类      | 有       | 类加载到卸载 | 方法区(元空间) |

## 4. 常量（final 变量）

使用 `final` 修饰的变量一旦赋值就不能修改。

```java
final double PI = 3.14159;
final int MAX_VALUE = 100;
// PI = 3.14;  // 编译错误: final 变量不能重新赋值
```

- 实例常量可在构造器中或声明时初始化；静态常量通常在声明时初始化。
- 常量命名规范：**全大写字母，单词间用下划线**。

## 5. 类型转换

### 自动类型转换（隐式转换）
小范围类型 → 大范围类型，自动进行：
```
byte → short → int → long → float → double
         ↑
        char
```
```java
int i = 100;
long l = i;        // 自动转换
double d = 10.5f;  // float -> double
```

### 强制类型转换（显式转换）
大范围 → 小范围，可能丢失数据或精度。
```java
double pi = 3.14;
int num = (int) pi;   // num = 3，小数截断

long big = 123456789L;
int small = (int) big; // 高位截断
```

## 6. 局部变量类型推断（Java 10+ 的 `var`）

从 Java 10 开始，局部变量声明可以用 `var` 推断类型，但仅在**局部变量且必须初始化**时使用。

```java
var name = "Java";      // 推断为 String
var list = new ArrayList<String>(); // ArrayList<String>
var sum = 10 + 20.5;    // double

// 注意：var 不能用于类的字段、方法参数、返回类型等。
```

## 7. 变量命名规则与规范

- **硬性规则（必须遵守）**：
    - 由字母、数字、下划线 `_`、美元符 `$` 组成。
    - 不能以数字开头。
    - 不能是 Java 关键字（如 `int`, `class`, `public`）。
    - 区分大小写。

- **约定俗成（强烈建议）**：
    - 变量名使用**小驼峰式**（camelCase）：`studentName`, `maxAge`。
    - 常量名全大写下划线：`MAX_SIZE`。
    - 见名知意，不要用 `a,b,x` 等无意义命名。

## 8. 变量在内存中的存储（简图）

- **栈**：存放局部变量（基本类型值或对象引用）。
- **堆**：存放实例变量和对象实体。
- **方法区**：存放静态变量、类信息等。

```java
public class Demo {
    int x = 10;               // 堆，跟随对象
    static int y = 20;        // 方法区
    public void show() {
        int z = 30;           // 栈
        String s = "test";    // 栈存引用，"test"在常量池(堆/元空间)
    }
}
```
