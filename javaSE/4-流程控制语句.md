# ☕ Java 流程控制语句详解

流程控制语句用于控制程序的执行顺序。Java 中的流程控制分为三大类：

· 顺序结构：程序从上到下逐行执行。
· 分支结构：根据条件选择执行某段代码。
· 循环结构：重复执行某段代码。

此外还有跳转语句用于改变循环或方法的执行流程。

---

1. 顺序结构

最基础的结构，代码按书写顺序依次执行。

```java
int a = 10;
int b = 20;
int sum = a + b;
System.out.println(sum);   // 先执行第一行，再第二行...
```

---

2. 分支结构

2.1 if 语句

单分支：条件为 true 时执行代码块。

```java
if (条件) {
    // 代码块
}
```

双分支：条件为 true 执行 if 块，否则执行 else 块。

```java
if (条件) {
    // true
} else {
    // false
}
```

多分支：多个条件依次判断。

```java
if (条件1) {
    // 1成立
} else if (条件2) {
    // 2成立
} else if (条件3) {
    // 3成立
} else {
    // 都不成立
}
```

示例：

```java
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}
```

注意事项：

· 条件表达式必须是 boolean 类型。
· 若代码块只有一条语句，可省略花括号（不推荐，易出错）。
· else 总是与最近的未配对的 if 结合（避免歧义请使用花括号）。

2.2 switch 语句

用于多值匹配的情况，比多重 if-else 更清晰。

语法（传统 switch）：

```java
switch (表达式) {
    case 值1:
        语句1;
        break;
    case 值2:
        语句2;
        break;
    // ...
    default:
        默认语句;
}
```

· 表达式的值可以是 byte, short, char, int，以及 String（JDK7+）、enum 枚举类型。
· 匹配到某个 case 后，执行后续语句，直到遇到 break 才跳出 switch。若没有 break 会穿透到下一个 case。
· default 可选，所有 case 都不匹配时执行。

示例：

```java
int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Other day");
}
// 输出：Wednesday
```

case 穿透示例（多个 case 共享代码）：

```java
int month = 8;
switch (month) {
    case 6:
    case 7:
    case 8:
        System.out.println("夏季");
        break;
    case 9:
    case 10:
    case 11:
        System.out.println("秋季");
        break;
    // ...
}
```

增强型 switch（Java 12+ 预览，14+ 正式）

使用箭头语法和表达式形式，避免穿透，更简洁。

箭头形式：

```java
switch (day) {
    case 1 -> System.out.println("Monday");
    case 2 -> System.out.println("Tuesday");
    case 3 -> System.out.println("Wednesday");
    default -> System.out.println("Other");
}
```

作为表达式返回值：

```java
String type = switch (day) {
    case 1, 2, 3, 4, 5 -> "工作日";
    case 6, 7 -> "周末";
    default -> "无效";
};
```

使用 yield 返回值（需要代码块时）：

```java
String result = switch (num) {
    case 1 -> "一";
    case 2 -> {
        String s = "二";
        yield s.toUpperCase();  // 返回"二"的大写
    }
    default -> "其他";
};
```

3. 循环结构

3.1 for 循环

适用于明确知道循环次数的情况。

语法：

```java
for (初始化; 循环条件; 迭代语句) {
    // 循环体
}
```

执行流程：

1. 执行初始化语句（只执行一次）
2. 判断循环条件，若为 true 执行循环体；否则退出
3. 执行迭代语句
4. 重复步骤 2

示例：

```java
for (int i = 1; i <= 5; i++) {
    System.out.println("i = " + i);
}
// 输出 1 2 3 4 5
```

· 初始化、条件、迭代三部分都可以省略，形成死循环：for (;;) { ... }
· 可以在初始化部分定义变量，作用域仅限于循环内。

增强 for 循环（for-each）：用于遍历数组或集合。

```java
int[] arr = {10, 20, 30, 40};
for (int num : arr) {
    System.out.println(num);
}

List<String> list = List.of("A", "B", "C");
for (String s : list) {
    System.out.println(s);
}
```

· 底层使用迭代器，简洁安全，但无法获取索引，且遍历时不能修改集合结构（会抛 ConcurrentModificationException）。

3.2 while 循环

适用于不确定循环次数，先判断条件再执行。

语法：

```java
while (条件) {
    // 循环体
}
```

示例：

```java
int i = 1;
while (i <= 5) {
    System.out.println("i = " + i);
    i++;
}
```

· 条件为 true 时执行，可能一次都不执行。
· 小心死循环：确保循环条件会变为 false。

3.3 do-while 循环

至少执行一次循环体，再判断条件。

语法：

```java
do {
    // 循环体
} while (条件);
```

示例：

```java
int i = 1;
do {
    System.out.println("i = " + i);
    i++;
} while (i <= 0);  // 条件为 false，但循环体已经执行了一次
// 输出：i = 1
```

· 注意 while 后面有分号。

4. 跳转语句

4.1 break

· 用于立即终止当前所在的循环或 switch 语句。
· 只能跳出一层循环。

```java
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;  // 当 i=5 时，结束整个循环
    }
    System.out.println(i);
}
// 输出：1 2 3 4
```

带标签的 break：可以跳出外层循环（不推荐滥用，影响可读性）。

```java
outer:
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (i * j > 4) {
            break outer;   // 跳出外层循环
        }
        System.out.println(i + "," + j);
    }
}
// 输出：1,1 1,2 1,3 2,1 2,2 （当 i=2,j=3 时 i*j=6>4，跳出 outer）
```

4.2 continue

· 跳过当前循环的本次剩余语句，进入下一次循环。
· 只能用于循环中。

```java
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;   // 跳过 i=3 的打印
    }
    System.out.println(i);
}
// 输出：1 2 4 5
```

带标签的 continue：跳过外层循环的某次迭代。

```java
outer:
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (j == 2) {
            continue outer;  // 跳过外层循环的本次剩余，直接 i++
        }
        System.out.println(i + "," + j);
    }
}
// 输出：1,1  2,1  3,1
```

4.3 return

· 用于结束方法的执行，并返回调用者。
· 可以带返回值（如果方法声明了返回类型）或不带（void 方法）。

```java
public static int add(int a, int b) {
    return a + b;   // 返回结果并结束方法
}

public static void printPositive(int num) {
    if (num <= 0) {
        return;     // 提前结束方法
    }
    System.out.println("正数：" + num);
}
```

5. 死循环与控制

当条件永远为真时形成死循环，通常需要配合 break 退出。

```java
while (true) {
    // 执行逻辑
    if (某个条件) {
        break;
    }
}
```

```java
for (;;) {
    // 语句
    if (条件) break;
}
```

6. 综合示例：猜数字游戏

```java
import java.util.Random;
import java.util.Scanner;

public class GuessNumber {
    public static void main(String[] args) {
        Random random = new Random();
        int target = random.nextInt(100) + 1;  // 1~100
        Scanner scanner = new Scanner(System.in);
        int guess;
        int count = 0;

        System.out.println("猜数字游戏（1~100）！");
        while (true) {
            System.out.print("请输入你的猜测：");
            guess = scanner.nextInt();
            count++;

            if (guess < target) {
                System.out.println("猜小了！");
            } else if (guess > target) {
                System.out.println("猜大了！");
            } else {
                System.out.println("恭喜你猜对了！共猜了 " + count + " 次。");
                break;
            }
        }
        scanner.close();
    }
}
```

---

总结

语句 用途 关键点
if-else 条件分支 条件必须是boolean；注意悬挂else
switch 多值匹配 支持byte/short/char/int/String/enum；注意break，JDK14+箭头语法
for 确定次数循环 三要素可省略；增强for遍历集合/数组
while 先判断后执行 可能一次都不执行
do-while 先执行后判断 至少执行一次；分号别忘
break 跳出循环/switch 仅跳出一层；标签可跳出外层
continue 跳过本次循环 标签可影响外层循环
return 结束方法 可带返回值

熟练掌握这些流程控制语句，你就能写出逻辑清晰、结构良好的 Java 程序。继续加油！
