# ☕ Java 运算符详解

运算符用于对变量和值执行操作。Java 提供了丰富的运算符，按功能可分为以下几类：

---

## 1. 算术运算符

用于基本的数学运算。

| 运算符  | 含义     | 示例            | 结果           |
|------|--------|---------------|--------------|
| `+`  | 加      | `5 + 3`       | `8`          |
| `-`  | 减      | `5 - 3`       | `2`          |
| `*`  | 乘      | `5 * 3`       | `15`         |
| `/`  | 除      | `5 / 2`       | `2`（整数相除得整数） |
| `%`  | 取模（取余） | `5 % 2`       | `1`          |
| `++` | 自增     | `a++` 或 `++a` | 加1           |
| `--` | 自减     | `a--` 或 `--a` | 减1           |

### 注意事项

**整数除法**：两个整数相除，结果仍为整数（截断小数部分）。
```java
System.out.println(5 / 2);   // 2，不是2.5
System.out.println(5.0 / 2); // 2.5（有一个是浮点数即可）
```

**取模运算**：结果的符号与被除数相同。
```java
System.out.println(10 % 3);   // 1
System.out.println(-10 % 3);  // -1
System.out.println(10 % -3);  // 1
```

**自增/自减的前后缀区别**：
```java
int a = 5;
int b = ++a;   // 前缀：先加再用，a=6, b=6

int x = 5;
int y = x++;   // 后缀：先用再加，y=5, x=6
```

## 2. 赋值运算符

| 运算符  | 等效写法        | 示例（a初始为10）    |
|------|-------------|---------------|
| `=`  | /           | `a = 5`       |
| `+=` | `a = a + b` | `a += 3` → 13 |
| `-=` | `a = a - b` | `a -= 3` → 7  |
| `*=` | `a = a * b` | `a *= 2` → 20 |
| `/=` | `a = a / b` | `a /= 3` → 3  |
| `%=` | `a = a % b` | `a %= 4` → 2  |

```java
int num = 10;
num += 5;     // 15
num -= 3;     // 12
num *= 2;     // 24
num /= 5;     // 4
num %= 3;     // 1
```

> 复合赋值运算符**隐含了强制类型转换**：
> ```java
> byte b = 10;
> // b = b + 5;   // 编译错误！int不能直接赋给byte
> b += 5;      // 正确，等效于 b = (byte)(b + 5)
> ```

---

## 3. 比较运算符（关系运算符）

结果始终为 `boolean` 类型（`true` 或 `false`）。

| 运算符          | 含义   | 示例       | 结果      |
|--------------|------|----------|---------|
| `==`         | 等于   | `5 == 3` | `false` |
| `!=`         | 不等于  | `5 != 3` | `true`  |
| `>`          | 大于   | `5 > 3`  | `true`  |
| `<`          | 小于   | `5 < 3`  | `false` |
| `>=`         | 大于等于 | `5 >= 5` | `true`  |
| `<=`         | 小于等于 | `5 <= 3` | `false` |

```java
int x = 10, y = 20;
boolean b1 = x > y;       // false
boolean b2 = x != y;      // true
boolean b3 = x == 10;     // true
```

> **注意**：`==` 对于引用类型比较的是**内存地址**，而非内容。比较对象内容应使用 `equals()` 方法。

---

## 4. 逻辑运算符

操作布尔值，返回布尔值。

| 运算符    | 含义   | 示例         | 说明               |
|--------|------|------------|------------------|
| `&&`   | 短路与  | `a && b`   | 两边都为true结果才为true |
| `\|\|` | 短路或  | `a \|\| b` | 至少一边为true结果为true |
| `!`    | 非    | `!a`       | 取反               |
| `&`    | 不短路与 | `a & b`    | 两边都会计算           |
| `\|`   | 不短路或 | `a \| b`   | 两边都会计算           |
| `^`    | 异或   | `a ^ b`    | 两边不同为true        |

### 短路与不短路的区别

```java
int a = 5;
// 短路：前面为false则后面不执行
boolean r1 = (a < 3) && (a++ > 0);  // false，a++未执行，a仍为5
System.out.println(a);  // 5

// 不短路：两边都执行
boolean r2 = (a < 3) & (a++ > 0);   // false，但a++执行了，a=6
System.out.println(a);  // 6
```

```java
int b = 10;
// 短路：前面为true则后面不执行
boolean r3 = (b > 5) || (b++ > 0);  // true，b++未执行，b仍为10

// 不短路：两边都执行
boolean r4 = (b > 5) | (b++ > 0);   // true，b++执行了，b=11
```

> 实际开发中，**短路运算符更常用**，效率更高，还能避免空指针异常：
> ```java
> if (str != null && str.length() > 0) { ... }  // 安全
> if (str != null & str.length() > 0) { ... }   // 若str为null会报错
> ```

## 5. 位运算符

直接对整数的二进制位进行操作，效率极高。

| 运算符 | 含义 | 示例（a=5, b=3） | 二进制过程 |
|--------|------|------------------|------------|
| `&` | 按位与 | `a & b` → 1 | 101 & 011 = 001 |
| `\|` | 按位或 | `a \| b` → 7 | 101 \| 011 = 111 |
| `^` | 按位异或 | `a ^ b` → 6 | 101 ^ 011 = 110 |
| `~` | 按位取反 | `~a` → -6 | ~00000101 = 11111010 |
| `<<` | 左移 | `a << 1` → 10 | 101 → 1010（末位补0） |
| `>>` | 右移（算术） | `a >> 1` → 2 | 101 → 010（高位补符号位） |
| `>>>` | 无符号右移 | `a >>> 1` → 2 | 高位补0 |

```java
int a = 5;   // 二进制: 0000 0101
int b = 3;   // 二进制: 0000 0011

System.out.println(a & b);    // 1   (0000 0001)
System.out.println(a | b);    // 7   (0000 0111)
System.out.println(a ^ b);    // 6   (0000 0110)
System.out.println(~a);       // -6  (1111 1010)
System.out.println(a << 2);   // 20  (0001 0100)  相当于乘以2²
System.out.println(a >> 1);   // 2   (0000 0010)  相当于除以2
```

### 位运算常见应用
```java
// 判断奇偶
boolean isEven = (num & 1) == 0;  // 位与1结果为0则是偶数

// 交换两数（异或）
int x = 10, y = 20;
x = x ^ y;
y = x ^ y;  // y 变成 10
x = x ^ y;  // x 变成 20

// 快速乘除2的幂
int m = n << 3;   // n * 8
int d = n >> 2;   // n / 4
```

## 6. 三元运算符（条件运算符）

是 Java 中唯一的**三元**运算符，作为 `if-else` 的简洁替代。

**语法**：`条件 ? 表达式1 : 表达式2`

```java
int score = 85;
String result = score >= 60 ? "及格" : "不及格";
System.out.println(result);  // 及格

// 嵌套使用（不推荐过度嵌套，可读性差）
int num = 10;
String type = num > 0 ? "正数" : (num < 0 ? "负数" : "零");
```

## 7. 字符串连接符 `+`

当 `+` 一边是字符串时，会执行字符串连接而非加法。

```java
System.out.println("Hello" + " World");   // Hello World
System.out.println("结果是：" + 10);       // 结果是：10
System.out.println("和是：" + 10 + 20);    // 和是：1020（从左到右拼接）
System.out.println("和是：" + (10 + 20));  // 和是：30（括号先算）
System.out.println(10 + 20 + "是结果");    // 30是结果（先加再拼）
```

---

## 8. `instanceof` 运算符

用于判断某个对象是否属于某个类或接口。

**语法**：`对象 instanceof 类型`

```java
String str = "Hello";
System.out.println(str instanceof String);  // true
System.out.println(str instanceof Object);  // true（String是Object子类）

Object obj = "World";
if (obj instanceof String s) {  // Java 16+ 模式匹配，直接转换
    System.out.println(s.toUpperCase());
}
```

## 9. 运算符优先级

高优先级先运算，同级别从左到右（赋值运算符从右到左）。

| 优先级 | 运算符 | 结合性 |
|--------|--------|--------|
| 最高 | `()` `[]` `.` | 左 |
| | `++` `--` `!` `~`（一元） | 右 |
| | `*` `/` `%` | 左 |
| | `+` `-` | 左 |
| | `<<` `>>` `>>>` | 左 |
| | `<` `<=` `>` `>=` `instanceof` | 左 |
| | `==` `!=` | 左 |
| | `&` | 左 |
| | `^` | 左 |
| | `\|` | 左 |
| | `&&` | 左 |
| | `\|\|` | 左 |
| | `? :` | 右 |
| 最低 | `=` `+=` `-=` 等 | 右 |

**记忆口诀**：单目 > 算术 > 移位 > 关系 > 位 > 逻辑 > 三元 > 赋值。

> **不确定优先级时，加括号最稳妥！** 括号不仅改变运算顺序，还能提高代码可读性。
